### Quickstart
Just run `npm start` in the root folder once requirements are met.
<br>
<br>

### Requirements
- Redis
- Mongodb
- [Nexus-Auth](https://github.com/Kaptard/nexus-auth) instance
<br>

**Note:** Make sure to set your preferences in /config/env.config.js
