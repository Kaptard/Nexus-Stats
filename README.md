### Quickstart
Just run `npm start` in the root folder once requirements are met.
<br>
<br>

### Requirements
- Redis
- Mongodb
<br>
<br>

### Default Ports
- client node: 3000
- api node: 3400
- db node: 3405
- redis: 6379
- mongo: 27017

You can change these at any time in /config/env.config.js
